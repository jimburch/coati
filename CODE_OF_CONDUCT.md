# Code of Conduct

Coati is an open-source project that welcomes contributors of all backgrounds and experience levels. To keep this community healthy, everyone participating — in issues, pull requests, discussions, and any other project space — is expected to follow these guidelines.

## Our standards

We aim for a community that is:

- **Respectful** — disagree with ideas, not people. Critique code, not character.
- **Constructive** — focus feedback on what can be improved and how.
- **Inclusive** — use welcoming language; assume good intent; avoid jargon where plainer words work.
- **Patient** — remember that contributors have different skill levels, time zones, and native languages.

Unacceptable behavior includes personal attacks, sustained disruption, publishing someone's private information without permission, and other conduct that a reasonable person would consider inappropriate in a professional setting.

## Scope

This Code of Conduct applies to all project spaces — the repository, issues, pull requests, discussions, and any official communication channels — and also when an individual is representing the project in public.

## Reporting

If you experience or witness behavior that violates this Code of Conduct, please report it by emailing `jim@showit.com`. Reports will be handled privately and in good faith. Maintainers are obligated to respect the privacy and security of the reporter.

Maintainers who do not follow or enforce this Code of Conduct in good faith may face consequences determined by other project leadership.

## Enforcement

Maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that violate this Code of Conduct. In serious or repeated cases, maintainers may temporarily or permanently ban a contributor from the project.
